# Notes

### Datatypes

Types of Datatypes:
	- Primitive
	- Non-Primitive

**Primitive**
	- integral
		- byte
		- short
		- int
		- long
	- floating point
		- float
		- double
	- char
	- boolean

**Datatype Table**

| Type | Size | Range | Default Value |

| short | 2 | -32768 to 32767 | 0 |
| int | 4 | -2147483648 to 2147483647 | 0 |
| long | 8 | - | 0 |
| float | 4 | ± 1.4E-45 to ± 3.4E+38 | 0.0f |
| double | 8 | ± 439E-324 to ± 1.7E+308 | 0.0d |

| boolean | ? | true/false | false |
